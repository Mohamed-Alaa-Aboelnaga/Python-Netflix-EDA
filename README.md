📊 **Data Analysis Project: Exploring Netflix Movies from the 1990s**

This is the **second project I worked on** as part of the **Data Analyst with Python** track on DataCamp. In this project, I analyzed Netflix movie data to explore trends in movies released during the 1990s.

🔍 In this project, I analyzed:
• Distribution of movie durations
• Number of movies released each year in the 1990s
• Genre patterns and short action movies

🛠 **Tools used:**
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

📈 **Key insights from the analysis:**
• Most movies in the 1990s had durations around 90–120 minutes.
• Movie releases increased in the late 1990s.
• Short action movies represented a smaller portion of the dataset.
